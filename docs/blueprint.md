# **App Name**: DoggyBoard

## Core Features:

- Visualización de métricas de progreso: Visualiza el progreso del entrenamiento del perro utilizando barras de progreso y métricas clave.
- Visualización de cursos activos: Muestra los cursos activos con porcentajes de finalización y enlaces de acceso rápido.
- Gráficos de rendimiento: Muestra gráficos de rendimiento semanales y mensuales para realizar un seguimiento de la curva de aprendizaje del perro.
- Gestión de tareas: Muestra una lista de tareas pendientes, logros recientes y análisis del tiempo de estudio.
- Botones de acción rápida: Proporciona botones de acceso rápido para clases, tareas y calificaciones.

## Style Guidelines:

- Color primario: Utiliza el color primario de la marca #3B71CA para los elementos interactivos principales, siguiendo el sistema de diseño CanineInsights Light.
- Color de fondo: Utiliza un fondo claro, #F8F9FA para el fondo primario y #FFFFFF para los elementos de la superficie como las tarjetas.
- Color del texto: Utiliza #212529 para el texto primario, #6C757D para el texto secundario y #ADB5BD para el texto terciario para garantizar la legibilidad.
- Familia de fuentes: Utiliza la fuente sans-serif 'Inter' para los encabezados y el texto del cuerpo para mantener una apariencia limpia y moderna.
- Diseño de tarjetas: Organiza el contenido en tarjetas modulares con sombras sutiles, adhiriéndote al diseño de tarjetas CanineInsights Light para un diseño estructurado.
- Espaciado: Aplica un espaciado consistente utilizando la unidad de espaciado del sistema de diseño de 8px, asegurando una presentación visual equilibrada y limpia.
- Iconos: Emplea iconos claros y consistentes utilizando el color #6C757D para representar varias funciones del panel.
